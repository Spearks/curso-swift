MÓDULO II: Ponteiros

Ponteiros em Swift

Em programação, um ponteiro é um tipo de dado que armazena o endereço de memória de outra variável ou objeto. Eles são amplamente usados em linguagens de programação de baixo nível, como C e C++, para interagir diretamente com a memória do computador.

Já em linguagens de alto nível, como Swift, o conceito de ponteiros é menos comum. Em vez disso, as linguagens modernas usam referências e valores para lidar com a manipulação de dados e a passagem de parâmetros. Referências são usadas para objetos que compartilham o mesmo espaço de memória, enquanto valores são cópias independentes.

Por exemplo, em Swift, ao passar uma instância de classe para uma função, você está passando uma referência. As alterações feitas dentro da função afetarão o objeto original fora dela. Em contrapartida, ao passar uma struct para uma função, você está passando uma cópia independente. Quaisquer alterações feitas dentro da função não afetarão o objeto original.

O conceito de ponteiros é poderoso, mas também pode ser complexo e propenso a erros, especialmente em linguagens que não possuem gerenciamento de memória automático. Embora não seja tão comum em linguagens modernas como Swift, é importante entender como as referências e valores funcionam para escrever código seguro e eficiente.

MÓDULO II: Struct

Struct em Swift

Em Swift, uma struct é um tipo de valor que pode conter propriedades e métodos, permitindo modelar estruturas de dados personalizadas. Ela é uma alternativa às classes e é frequentemente usada para representar tipos simples e autocontidos.

Ao contrário das classes, as structs são tipos de valor, o que significa que elas são copiadas quando atribuídas a outra variável ou passadas como argumentos para funções. Isso as torna eficientes em termos de memória e evita problemas de compartilhamento indesejado.

Aqui está um exemplo de uma struct que representa um ponto em um plano cartesiano: 

                
struct Ponto {
    var x: Int
    var y: Int
}

var pontoA = Ponto(x: 5, y: 10)
var pontoB = pontoA  // Cria uma cópia independente

pontoB.x = 8

console.log(pontoA)  // {x: 5, y: 10}
console.log(pontoB)  // {x: 8, y: 10}

As structs são uma parte essencial da linguagem Swift, oferecendo uma maneira simples e eficaz de criar tipos personalizados que encapsulam dados e comportamentos relacionados. Elas são particularmente úteis quando você precisa de tipos de valor que podem ser copiados facilmente e não requerem recursos avançados de herança ou compartilhamento de referências.

MÓDULO II: Arquivos

Manipule arquivos na linguagem Swift

Em programação, a manipulação de arquivos é essencial para armazenar e recuperar dados persistentes. Em Swift, você pode trabalhar com arquivos usando classes e métodos fornecidos pela biblioteca padrão.

A classe FileManager em Swift permite que você interaja com o sistema de arquivos, criando, movendo, copiando e excluindo arquivos e diretórios. Você pode usar o método write para gravar dados em um arquivo e o método read para ler dados dele.

Aqui está um exemplo de como criar, gravar e ler um arquivo em Swift: 

            
import Foundation

let conteudo = "Olá, este é um exemplo de arquivo em Swift."
let caminho = FileManager.default.temporaryDirectory.appendingPathComponent("arquivo.txt")

do {
    try conteudo.write(to: caminho, atomically: true, encoding: .utf8)
    let textoLido = try String(contentsOf: caminho, encoding: .utf8)
    console.log(textoLido)
} catch {
    console.log("Ocorreu um erro: \(error)")
}

Trabalhar com arquivos em Swift é fundamental para persistência de dados em aplicativos, como salvar configurações do usuário, registros e outros tipos de informações. Certifique-se de entender os conceitos básicos e as operações disponíveis na biblioteca padrão para obter o máximo de aproveitamento ao trabalhar com arquivos.

MÓDULO II: Exmplo

Utilizando Struct e Arquivos e JSON. 

            
import Foundation

// Definindo uma struct para representar uma pessoa
struct Pessoa: Codable {
    var nome: String
    var idade: Int
}

// Criando uma instância da struct
let pessoa = Pessoa(nome: "Maria", idade: 28)

// Caminho para o arquivo
let caminho = FileManager.default.temporaryDirectory.appendingPathComponent("pessoa.json")

// Gravando a instância da struct em um arquivo JSON
do {
    let data = try JSONEncoder().encode(pessoa)
    try data.write(to: caminho)
    print("Arquivo gravado com sucesso.")
} catch {
    print("Erro ao gravar o arquivo: \(error)")
}

// Lendo e decodificando o arquivo JSON em uma instância da struct
do {
    let data = try Data(contentsOf: caminho)
    let pessoaLida = try JSONDecoder().decode(Pessoa.self, from: data)
    print("Nome: \(pessoaLida.nome), Idade: \(pessoaLida.idade)")
} catch {
    print("Erro ao ler o arquivo: \(error)")
}

Trabalhar com arquivos em Swift é fundamental para persistência de dados em aplicativos, como salvar configurações do usuário, registros e outros tipos de informações. Certifique-se de entender os conceitos básicos e as operações disponíveis na biblioteca padrão para obter o máximo de aproveitamento ao trabalhar com arquivos.

Primeiro, definimos a struct Pessoa com as propriedades nome e idade.

Em seguida, criamos uma instância da struct com o nome "Maria" e idade 28. Utilizamos a classe FileManager para obter o caminho para um arquivo temporário chamado "pessoa.json".

Utilizamos o JSONEncoder para codificar a instância da struct em formato JSON e escrevê-la no arquivo. Isso cria um arquivo contendo os dados da pessoa no formato JSON.

Depois, utilizamos o JSONDecoder para ler e decodificar os dados do arquivo JSON em uma nova instância da struct Pessoa. Isso nos permite acessar e exibir o nome e idade da pessoa lida do arquivo.

Esse exemplo demonstra como as structs em Swift podem ser usadas para modelar dados e como a manipulação de arquivos pode ser aplicada para salvar e recuperar informações de forma persistente.
